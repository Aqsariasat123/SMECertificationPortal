# Brand Voice & UX Writing Guide
## SME Certification Portal - Phase 1

---

## Purpose

This guide exists to ensure **long-term consistency** as the platform scales. It is a **reference**, not a design constraint.

As we continue building out the platform, this document serves as a shared reference going forward. It is not meant to introduce new scope or changes to what's already approved â€” simply a baseline to help keep copy, labels, CTAs, and visual cues consistent as new screens and flows are added.

---

## Core Principles

The core idea is straightforward:

- **The platform should read like a certification authority**, not a marketplace or startup.
- **Language should stay certification-first**, registry-focused, and action-led.
- **Visual and copy choices should reinforce trust**, clarity, and institutional credibility.

> Please treat this guide as a reference point when working on future UI, copy, and UX decisions. If anything ever feels unclear or conflicting, align with the team quickly.

---

## Core Positioning
**Certification-first platform** - NOT a marketplace, NOT a fundraising platform.

## Terminology Rules

### AVOID These Terms (Legacy/Generic)
| Do NOT Use | Reason |
|------------|--------|
| Registration | Generic, not certification-focused |
| Create Account | Identity-based, not action-led |
| Sign Up | Marketplace language |
| Investor | Phase 1 focuses on certification only |
| Fundraising | Not in scope for Phase 1 |
| Investing | Not in scope for Phase 1 |
| Pledging | Not in scope for Phase 1 |
| Deal-making | Marketplace language |
| I am a SME / I am an Investor | Identity-based framing |

### USE These Terms (Certification-First)
| Use This | Context |
|----------|---------|
| Start Certification | Primary CTA |
| Complete Setup | Final step button |
| Setup Complete | Success message |
| Secure Your Access | Heading for user credentials step |
| Access Registry | Secondary CTA for browse path |
| Certification | Core action |
| Verification | Process description |
| Credibility | Value proposition |
| Registry Access | Benefit |
| Official UAE SME Registry | Key value prop |

## Screen-by-Screen Terminology

### Account Type Selection (Step 1)
| Element | Correct Copy |
|---------|--------------|
| SME Card Heading | Certify a Business |

| SME Button | Start Certification |
| Browser Card Heading | Browse Certified Businesses |
| Browser Card Subtext | Access the registry of verified SMEs. |

### Company Information (Step 2 - SME)
| Element | Correct Copy |
|---------|--------------|
| Heading | Company Information |
| Subtext | Enter your business details to get started with certification. |
| Button | Save & Continue |

### Personal Details (Step 3 - SME / Step 2 - Browser)
| Element | Correct Copy |
|---------|--------------|
| SME Heading | Personal Details |
| SME Subtext | Set up your login credentials to complete setup. |

|---------|--------------|
